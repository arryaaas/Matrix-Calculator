# Matrix-Calculator

Some facts about this project:

- Completely written in Kotlin

## Preview App

![](MatrixCalculatorPreview.png)

## Features

The Matrix Calculator app lets you to calculate:
- Addition
- Subtraction
- Multiplication
- Determinant
- Trace
- Inverse
- Transpose
- Scalar Multiplication
